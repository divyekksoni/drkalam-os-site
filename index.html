An OS that takes flight

DrKalamOS is an open‑source, offline‑first operating system built for Indian schools, government offices, and beyond. Inspired by Dr. A.P.J. Abdul Kalam's vision of self‑reliance.

Core Features

Lightweight & Fast

Runs on 2 GB RAM and 10+ year old hardware. Optimised Xfce desktop for speed.

Alpha

22+ Indian Languages

Full support for Indic scripts, fonts, and phonetic typing (Inscript / M17n).

Alpha

Offline-First

Works without internet. Local content, P2P sharing, and offline quizzes.

Alpha

Education Ready

DIKSHA NCERT content, teacher dashboard, and exam mode (planned).

Planned

Secure by Design

Read‑only root with overlay (kiosk mode), UFW firewall, and OpenSSH.

Planned

RISC‑V Ready

Future support for Indian‑made RISC‑V chips and low‑cost ARM devices.

Planned

Project Orbit Tracker

Join the Launch Sequence

DrKalamOS is built by the community, for the community. Everyone is welcome — developers, designers, educators, translators, and testers.

Developers

Pick a "good first issue" on GitHub. Help with build scripts, features, or bug fixes.

Translators

Add support for more Indian languages. Translate UI, docs, and content.

Educators

Test the OS in your classroom. Share feedback, suggest features, and pilot the OS.

Designers

Improve the desktop theme, icons, and branding. Make DrKalamOS beautiful.

Testers

Run the OS on different hardware. Report bugs and help us improve stability.

Advocates

Spread the word. Connect us with schools, colleges, and government bodies.
